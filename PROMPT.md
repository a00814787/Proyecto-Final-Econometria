# Prompt de Desarrollo — Proyecto Final Econometría

## Herramienta utilizada
Claude (Anthropic) — Asistente de IA para desarrollo del proyecto

## Prompt principal utilizado

"Quiero elaborar un proyecto final para una materia de econometría 
que construya un portafolio de inversión de $100,000,000 USD en 
HowMarketWorks aplicando las siguientes herramientas econométricas:

- Análisis Exploratorio de Datos (EDA)
- Análisis de Correlación (Pearson)
- Regresiones Lineales Simples y Múltiples
- Validación de supuestos: normalidad, homocedasticidad (Breusch-Pagan, 
  Goldfeld-Quandt), autocorrelación (Durbin-Watson) y multicolinealidad (VIF)
- Series de Tiempo ARIMA + GARCH (Google Colab / Python)
- Panel Data con efectos fijos (PanelOLS)

Datos: Refinitiv (diarios, 3 años). Software: R/R Markdown y Python/Colab.
Resultado final: portafolio de 8 activos con pesos justificados 
econométricamente."
